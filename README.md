# SimpleBlog

A Django-powered blogging application built as a first step into Django development. SimpleBlog demonstrates core Django concepts including models, views, templates, and user authentication.

Experience SimpleBlog live here: [https://django-msz3.onrender.com/](https://django-msz3.onrender.com/)

- 📝 Create, read, and manage blog posts
- 👤 User authentication and authorization
- 🎨 Clean and responsive user interface
- 📱 Mobile-friendly design
- 💬 Interactive blog experience

## 🎯 Purpose

SimpleBlog serves as an educational project for learning Django fundamentals:
- Model creation and database relationships
- Django Views and URL routing
- Template rendering
- User authentication systems
- Static files management (CSS, images)
